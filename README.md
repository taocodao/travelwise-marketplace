# TravelWise Marketplace
AI Agent Monetization Platform with x402 Micropayments and ERC-8004 Reputation
## Quick Start
See [SETUP.md](./SETUP.md) for complete setup instructions.
## Architecture
- **Frontend**: Next.js 14 (TypeScript)
- **Backend**: Node.js + Express (TypeScript)
- **Smart Contracts**: Solidity (ERC-8004)
- **MCP Servers**: Google Maps, Weather, Travel Agent
- **Blockchain**: Base Sepolia (Testnet)
- **Payment**: x402 Protocol with USDC
